# URGED Color Schemes
This file describes the color coding used in this project:

## LCZs:

The official color scheme uses:

* **Red tones** (dark → bright) for compact built types (LCZ 1-3)
* **Orange tones** (dark → light) for open built types (LCZ 4-6)
* **Yellow** for lightweight lowrise (LCZ 7)
* **Gray tones** for large lowrise and heavy industry (LCZ 8, 10)
* **Peach** for sparsely built (LCZ 9)
* **Green tones** for natural land cover (LCZ A-G)
